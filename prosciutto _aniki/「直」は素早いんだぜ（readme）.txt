・はじめに
このファイルをダウンロードして頂き、誠に有難うございます
このキャラクターは漫画「ジョジョの奇妙な冒険」の
第五部に登場する敵役のプロシュート兄貴をイメージして作られています。
システムは三部格ゲー「未来への遺産」のものを一部流用しております。
また、このキャラクターにはデフォルトでAIが同梱されています。

・キャラクター紹介

イタリアのギャング組織『パッショーネ』の暗殺チームのひとり
頼れる兄貴分的な性格で、弟分でコンビのペッシに慕われている
ボスの正体に繋がる（であろうと推測される）トリッシュを奪うために
その護衛の任務に当たっていたブチャラティ達と戦うことになる。

　スタンド『ザ・グレイト・フルデッド』
破壊力―B スピード―E 射程距離―列車一本分程度は十分
持続力―Ａ 精密動作性―Ｅ 成長性―C

生物を無差別に、身も心も記憶力も老化させる能力。
体温の微妙な上昇差によって男女の区別をする。

・コンボ補正
このキャラクターはコンボ数に比例して攻撃力に補正が掛かるようになっています。
コンボ数が増えるほどに、技単発での威力は低下します。

■システム面解説■
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
{スタンドモード}
aボタンでオンオフの切り替えが可能です。
スタンドモード時には以下の特性が追加されます
・通常攻撃が全て変化
・必殺技とスーパーコンボの性能も変化
・ガード時に体力を削られなくなる
・移動速度がやや低下
・ダウンしなくなる
※ジャンプ回数は一回のまま
…他にも細かい点でいろいろ差異があります。

{スタンドゲージ}
スタンドモード時にダメージを受けた場合に減少し
これが空になると強制的にスタンドモードが解除され、さらに大きな隙が生まれてしまいます。
スタンドモードがオフの場合は自動的に回復します。

{老化ゲージ}
特定の攻撃を使用、もしくはHITさせた場合に最高で八段階まで上昇します。
試合開始時は貧弱なマンモーニ（ママっ子）並のダメージしか与えられませんが
老化ゲージが上昇するにつれて相手に与えるダメージも強化されていきます。
どの技でゲージが上昇するのかについては下記参照。

{プッツンキャンセル}
超必殺以外の攻撃動作を、1ゲージ消費してcボタンでキャンセル可能
コンボや攻撃後の隙消しに活用できます

{未来への遺産から以下のシステムを搭載}
・アドバンシングガード
・ガードキャンセル攻撃
・回り込み

{その他}
・OFF時通常攻撃はスタンド発現攻撃でキャンセル可能
・空中弱・中攻撃は空中強攻撃でキャンセル可能(その場合少し浮く)

必殺技
タッチオブグレイ　D, DF, F + x/y/z 【ON OFF共通】
グレイト・フルデッド（以下GF）が前方に突きを三連打する技。
リーチが長く牽制にも使えるが、外すとスキが大きめ。空中でも使用可能
ON時では、ボタン入力によって一定距離を前進するようになっている
技名はロックバンド『グレイトフル・デッド』の楽曲名から命名しました

朽ち果てろ！ F, DF, D, DB, B + x/y/z 【ON OFF共通】（HIT時老化ゲージ+1）
近距離の相手を掴んで老化させる技、老化ゲージを一つ溜めることができる。
掴める範囲は狭いがガード不能で、当てた場合のリターンも大きい。
ON時では、相手を掴める範囲が広くなっている

「ブッ殺す」と心の中で思ったなら！DF＋z 【ON OFF共通】
足元に向けて発砲する技。スキが大きめでリーチにも優れていないが
弾丸が下段判定で、なおかつダウン中の相手にもHITする特性がある
ON時では、発砲と同時にスタンドが攻撃を行うように強化されている。
また、同時にxかyボタンを入力する事でスタンドの攻撃が変化する

のろいぞ！　B, D, DB + x/y/z  【OFF時専用】（HIT時老化ゲージ+1）
カウンター技。ボタン入力で防御する位置が変化する。
カウンターに成功した場合は相手を掴んで老化させることが出来る。
また、カウンター成功時のつかみ動作は回り込みでキャンセルも可能。
相手との距離が遠い場合や、相手のスキが特に大きい場合は有効かも

ダークスター　DD + x/y/z  【ON時専用】
上方へ飛び上がる対空技。ボタン入力で上昇速度が変化する。
真上に飛ぶため横方向への判定が弱めだが、発動時に少し無敵がつく
ちなみにコレも楽曲から技名を命名
見た目が地味な技なので、名前負けしてるかもしれない
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
スーパーコンボ　※全てスタンドON OFF問わず使用可能

偉大なる死　　D, DB, B + xy/yz/xz/b　（時間経過で老化ゲージ+2）
GFが大量の老化ガスを噴出する技。近距離の相手を吹っ飛ばすことができる。
与えるダメージは微々たる物だが、技を出し切ると老化ゲージを２つ溜めることができる。
ただしGFが消えるまでの時間が長く、相手からの妨害を受けやすい弱点もある。

直は素早いんだぜ！　　D, DF, F + xy/yz/xz/b（HIT時老化ゲージ+1or2）
相手を打ち上げてから掴み、直接老化させる技。
リーチは短いが、コンボにつなぎやすい特徴を持っている。
また、この技を素で当てた場合は老化ゲージを二つ溜めることができるが
コンボに組み込んだ場合は一つしか溜めることが出来ない。

直に叩き込む！　　D, DF, F + xyz（HIT時老化ゲージ+2）
『直は素早いんだぜ！』と同様に、
相手を打ち上げてからダウンした相手にスタンド能力を叩き込む技。
発動にはパワーゲージを２消費するが威力は非常に高く
コンボに組み込んでも必ず老化ゲージを二つ奪える他
『直は素早いんだぜ！』よりもコンボ補正が掛かりにくい長所もある
