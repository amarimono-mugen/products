通常技

弱攻撃(L)　yボタン

中攻撃(M)　aボタン

強攻撃(H)　bボタン
;----------------------------------------------------------------------------
特殊技

小ジャンプ　飛んですぐ↓

回り込み　x

軽やかな身のこなし　F, D, DF ＋ z　B, D, DB ＋ z　0.5ゲージ消費。レバー方向へ投げ以外無敵状態でスライド移動する。相手との距離を詰めたい場合や、逆に距離をとりたい場合にも

アドバンシングガード　ガード中に x + y + z　or z or b

ガードキャンセル攻撃　ガード中に F, D, DF ＋ 攻撃ボタン

ダウン回避　攻撃ボタン二つ同時押し

波紋の呼吸　c パワーゲージを徐々に増加する

座ったままの跳躍　ｓ　回避になる…のだろうか

投げ　相手の近くで　F＋b or B＋b コミックス３巻で所持してた傘で殴る。

傘で殴る　　a 傘で殴る技。一切のキャンセルが効かないが対空にはなる...かもしれない

ズームパンチ F + a
前方に伸びるパンチを放つ特殊技。ヒット時に崩れ落ちダウンを誘発する

生命磁気への波紋疾走　空中で F + c 発動時に0.2ゲージ消費。
葉っぱを波紋で集めて滑空する技。
空中通常技のヒットorガード時にキャンセルして出せる。敵の攻撃を飛び越したり、ガードを崩したい場合に有用。

;----------------------------------------------------------------------------
必殺技

スーパーズームパンチ　　D, DF, F ＋ x/y/b　異常なほどリーチが長いパンチ。弱中強の順にリーチが増えるが隙も増加する

仙道波紋　　D, DB, B ＋ x/y/z ジョセフのとほぼ同じ

仙道波蹴(ウェーブキック)　　F, D, DF ＋ x/y/z　対空技。弱中強の順にリーチが増えるが隙も増加する

波紋乱渦疾走(トルネーディ・オーバードライブ)　　B, D, DB ＋ x/y/z　真上に飛び上がった後、回転しながら落下しつつキックを放つ。
落下時は中段判定で、着地後の蹴りは下段判定。
ボタンで移動距離がアップするが隙も大きくなる。空中でも使用可能。
;----------------------------------------------------------------------------
EX必殺技(0.25ゲージ消費)

スーパーズームパンチ　　D, DF, F ＋ c　強版のリーチで発生が超早い。ただし後隙は気持ち多め。

仙道波紋　D, DF, F ＋ c　弱版の発生速度 + 強版のふっとばし効果

仙道波蹴(ウェーブキック)　F, D, DF ＋ c　移動距離UP + 出だしの無敵時間増加。

波紋乱渦疾走(トルネーディ・オーバードライブ)　B, D, DB ＋ c　降下角度をより鋭く + 出だしに無敵が追加
;----------------------------------------------------------------------------
ゲージ技
波紋カッター　　D, DF, F + 2ボタン or x
1ゲージ消費。切り裂きジャックに放った技。遠距離用

るオオオオオ！　　D, DB, B + 2ボタン or x
1ゲージ消費。カエルを潰すかと思わせて岩を砕いたあのパンチ。砕いた岩盤に判定があるぞ。ダウン時の相手にもヒットする。

山吹色の波紋疾走　　F, D, DF ＋ 3ボタン or  z
2ゲージ消費。モーションと効果はジョセフの方と同じだが、威力が上がり２ゲージ技になっている。
ヒット後は背景にツェペリさんの思い出が流れる。
