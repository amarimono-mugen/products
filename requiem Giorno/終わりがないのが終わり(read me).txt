・はじめに
このファイルをダウンロードして頂き、誠に有難うございます
このキャラクターは漫画「ジョジョの奇妙な冒険」の第五部に
登場する、『矢』の力によってレクイエムの能力を手に入れた
ジョルノ・ジョバァーナをイメージして作られています。
システムは三部格ゲー「未来への遺産」のものを一部流用しております。
また、このキャラクターにはデフォルトでAIが同梱されています。
詳しくはAI側のreadmeで

アーケード版ジョジョ格ゲー(未来への遺産)を基にしています。

※注意！
1.製作者は格闘ゲームにあまり詳しくありません。
よって、性能面でのバランスに難が有る可能性があります。

2.キャラクターの能力の解釈や技の性能は、製作者個人の独断で決定しています。
人によっては原作でのイメージと製作物のイメージが合致しない可能性があります。

・制限事項
改変の制限は特に設けません。AI製作などもOK(むしろ大歓迎)です。
動画での使用や、サイトでの紹介もOKです。(こちらも大歓迎)
二次配布は許可しません
商用利用は（する人いないと思うけど）不可です
また、動画使用の際は出来る限り最新版での仕様をお願いします。
バグとか不具合が出る可能性が高いので。

性能解説
通常版ジョルノをベースに作られていますが、性能は全く別物になっています
基本的にはアッパー調整で、体力のみオリジナルの80％に抑えられています。
一応、理不尽すぎる性能にはならないように気を配りましたが
性能面で見れば、やはり色々ブッ飛んでるかと思います。

また、12Pカラーは特殊な仕様になっており、スタンドON時に
常時『ドコヘモムカウコトハナイ…』のカウンター状態になり
さらに飛び道具と投げに対して常に無敵が付与され攻撃力が2倍に上昇します
ぶっちゃけまともな方法では倒せなさそうです。ネタとして捉えてください。

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
{スタンドモード}
aボタンでオンオフの切り替えが可能です。
スタンドモード時には以下の特性が追加されます
・通常攻撃が全て変化
・必殺技とスーパーコンボの性能も変化
・ガード時に体力を削られなくなる
・2段ジャンプと空中ダッシュが可能に
・ダウンしなくなる
…他にも細かい点でいろいろ差異があります。

{スタンドゲージ}
スタンドモード時にダメージを受けた場合に減少し
これが空になると強制的にスタンドモードが解除され、さらに大きな隙が生まれてしまいます。
スタンドモードがオフの場合は自動的に回復します。

{空中ダッシュ}
スタンドオン時に、空中でレバーを前後どちらかに二度素早く入れることで
入れた方向に向かってダッシュすることが出来ます。
また、ダッシュ時にレバーを上下どちらかに入れることで高度を調節できます

{プッツンキャンセル}
超必殺以外の攻撃動作を、1ゲージ消費してcボタンでキャンセル可能
コンボや攻撃後の隙消しに活用できます

{未来への遺産から以下のシステムを搭載}
・アドバンシングガード
・ガードキャンセル攻撃
・回り込み(ON時でも可)

{その他}
・OFF時通常攻撃はスタンド発現攻撃でキャンセル可能
・空中弱・中攻撃は空中強攻撃でキャンセル可能(その場合少し浮く)

補正
・コンボ補正
このキャラクターはコンボ数に比例して攻撃力に補正が掛かるようになっています。
コンボ数が増えるほどに、技単発での威力は低下します。
・根性値
相手との体力差に応じて与えるダメージが変動します。
相手より体力が多い場合は与えるダメージが少なくなり
逆に相手より体力が少ない場合は与えるダメージが多くなります

{特殊カラー}
11Pカラー※キャラ選択時にs+yで選択可能
・スタンドON時に、常に自動カウンター状態
・スタンドON時の攻撃力1.5倍
・『ドコヘモムカウコトハナイ…』『終わりがないのが無いのが終わり』が使用不可

12Pカラー※キャラ選択時にs+zで選択可能

・常に自動カウンター状態かつ、カウンター成功時に一撃必殺技に移行
・スタンドON時の攻撃力3倍
・『ドコヘモムカウコトハナイ…』『終わりがないのが無いのが終わり』が使用不可
・常にSonの為、タンデムアタックが使用不可
・サソリがガード不可に
・常時パワーゲージが大回復

必殺技（soff時）

無駄無駄　　D, DF, F ＋ x/y/z　
GERが前方に飛び出して相手を連打する技。
発生、威力、削りなどがかなり優秀。
ただし出たらすぐには消えないので、相手にかわされると危険。

エアシュート　　D, DF, F ＋ x/y/z（空中で）
GERが斜め下方向に蹴りを放つ技。
ボタン入力によって、発生までの時間を調整できる

生命よ！　F, DF, D, DB, B ＋ x/y/z 
地面から木の根を成長させて攻撃する技。
これは通常版とあまり変わらない。

サソリ射出　D, DB, B ＋ x/y/z　
飛び道具(サソリ)を前方に高速で打ち込む。
通常版のトビウオより速度、威力で優れる。

レクイエムの片鱗　B, D, DB ＋ x/y/z　
カウンター技。成功時は壁バウンドからの追撃が可能
;----------------------------------------------------------------------------
必殺技（son時）

無駄無駄　　D, DF, F ＋ x/y/z（空中可）
スタンドによる高速のラッシュ。
弱中強で移動速度が変化し、ボタン連打で攻撃回数が増加する。
また、空中での使用も可能。

エアシュート　　D ＋ z（空中のみ）
ヒットした場合に、二回まで同コマンドの追加入力が可能。
その場合、最後の攻撃で相手を大きく吹っ飛ばせる。

樹木生成　　F, DF, D, DB, B ＋ x/y/z
地面から樹木を成長させて攻撃。ボタンによって出現位置が変化する
相手を情報に突き上げるので、追い討ちも可能。　空中ガード不可

サソリ射出　D, DB, B ＋ x/y/z(空中可)
空中でも使用可能。また、空中ではボタンによって射出角度が変化する。　

ドコヘモムカウコトハナイ…　c(空中可)※要0.5ゲージ
少し特殊なカウンター技。
発動時に、範囲内の相手が攻撃の意思を見せた場合、(movetype = Aの場合)
その行動を無効にして隙を作り出すことができる。
加えて範囲内に入れば飛び道具をも無効化することができ、相手に投げられることもない。
また、単発の飛び道具を無効化した場合は、消費した分のゲージを回収でき、
複数発の飛び道具を無効化すれば、消費した以上のゲージを回収できる。
その性質上カウンターが成功しやすく、範囲も広めでかなり強い技。

スーパーコンボ（soff時）
無駄無駄ラッシュ　　D, DF, F, xy
お馴染みのラッシュを放つスパコン　出始めに無敵があるため、割り込みにも使える
また最後の一撃は相手を強く吹っ飛ばす。やっぱり削りもスゴイ

木の根（スパコン）　　D, DB, B, xy
地面から木の根を前方に向けて連続で生み出す。ヒット後は追撃も可能　
;----------------------------------------------------------------------------
スーパーコンボ（son時）
無駄無駄ラッシュ　　D, DF, F, xy(空中可)
お馴染みのラッシュを放つスパコン　off時よりも威力が高い
出始めに無敵があるため、割り込みにも使える
空中でも使用可能であり、その場合はレバー入れで速度や高度の調整ができる

木の根（スパコン）　　D, DB, B, xy
地面から木の根を連続で生み出す。
ON時のみの特性として、相手の位置をサーチして発生するように強化されている。

無駄無駄無駄無駄　　B, DB, D, DF, F, xy　※要2ゲージ
前方に蹴りを繰り出し、ヒットした場合敵を拘束して蹴りのラッシュを放つ。
ゲージ消費が激しいが、威力が非常に高い。

終わりが無いのが終わり　B, D, DB xy ※要3ゲージ
３ゲージ以上のときに発動可能なカウンター技。
成功すれば一撃で相手を葬り去ることができる。
ただし、カウンターを構えるだけで3ゲージ消費する高燃費な技でもある。
また、この技で倒した相手は(最終ラウンド時のみ)画面外へ吹っ飛んで消える。
ちなみにこの技でも飛び道具を無効化可能だが、まず赤字になるのでオススメしない
